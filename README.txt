Karp Synthesizer by Nathan Blair (@nthnblair)

A software synthesizer implementing the Karplus-Strong synthesis algorithm. 

 - MIDI polyphonic up to 20 voices and compatible with MPE.  
 - Available in VST3 and AU formats. 

Install Locations: 
 - /Library/Audio/Plug-Ins/VST3/Karp.vst3
 - /Library/Audio/Plug-Ins/Components/Karp.component

To uninstall this software, move the installed Karp.vst3 and Karp.component files to the trash.
